# MyHouse

背景：今年房地产疯涨，国家出台最严厉限购，市场急速降温，预计房地产一年内横盘阴跌，之后可能会慢慢上涨。
项目的目的是抓取网上二手房的数据进行分析，了解房价趋势，为购房提供的决策分析，以最小的成本完成安居乐业的愿望。

# 实现思路

收集数据 -> 分析数据

# 项目整体介绍

项目整体分为以下个模块

1. 数据收集

专门用于收集数据，用于进一步的分析。

2. 数据分析

对收集的数据进行分析，并以图表的方式进行展示。

3. 预警模块

设置有意义的预警点，对抓取数据进行实时预警和关注。
